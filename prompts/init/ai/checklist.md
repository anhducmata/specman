# AI Coding Checklist

> ⚠️ DRAFT — Checklist for AI tools before submitting code changes.

## Pre-Coding
- [ ] Read relevant specs
- [ ] Check domain rules
- [ ] Check ADRs
- [ ] Search solved cases

## During Coding
- [ ] Follow coding rules
- [ ] Follow testing rules
- [ ] Handle errors properly
- [ ] Do not violate domain rules

## Post-Coding
- [ ] Run tests
- [ ] Check for secrets
- [ ] Verify no spec conflicts
- [ ] Consider creating a solved case
